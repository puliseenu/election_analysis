═══════════════════════════════════════════════════════════════════════

auth.db (SQLite)
  • User database
  • Auto-created on first run
  • Contains:
    ├─ users table (accounts, passwords)
    ├─ audit_log table (action history)
    └─ sessions table (future use)
  • Features:
    ├─ User registration
    ├─ Password storage (hashed)
    ├─ Approval workflow
    └─ Activity logging
  • To reset: Delete file and restart app
