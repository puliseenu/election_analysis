❓ COMMON ISSUES & SOLUTIONS

Q: "Cannot reach 192.168.0.165:8050"
A: • Check WiFi: Both devices on same network?
   • Firewall: Is Python allowed through firewall?
   • Dashboard running: See terminal showing "Dash is running"?
   • Correct IP: Try ipconfig | findstr IPv4

Q: "ngrok fails with 'authentication failed'"
A: • Did you set NGROK_AUTHTOKEN?
   • Check: echo $env:NGROK_AUTHTOKEN
   • Restart PowerShell after setting token
   • Token from: https://dashboard.ngrok.com

Q: "Port 8050 already in use"
A: Kill existing process:
   Get-Process python | Stop-Process -Force
   Then restart: python main_dashboard.py

Q: "Dashboard is slow on large filters"
A: • Filter to specific state/year
   • Or use State-wise summary tables
   • Dashboard is fast for <10,000 records
